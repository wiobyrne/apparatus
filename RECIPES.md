# Recipes

Short page recipes for common Apparatus layouts.

## Archive Page

- Eyebrow
- H1
- Short lede
- Record block
- Archive list grouped by year or section

Use this for `/writing/` and similar index pages.

## Publication Landing Page

- Eyebrow
- H1
- Lede
- Record block
- Offer sections
- Archive or issue list

Use this for `/newsletter/` and publication front doors.

## Post Page

- Eyebrow
- H1
- Subtitle / description
- Optional lede
- Record block
- Body
- Optional `see also`
- Prev / next navigation
- Colophon

Use this for essays, newsletters, and long-form posts.

## Static Page

- Eyebrow
- H1
- Body

Use this for about, contact, work with me, and other utility pages.

## Notes

- Keep the layout vertical unless a real reading reason creates another structure.
- Keep the record block factual.
- Let the body do the work after the top has oriented the reader.
